Команды в терминале Git

  • git init - создать git репризиторий
  • git config --global user.name "..."
  • git config --global user.email "...
  • git status - статус гита
  • git add -A - добавить все файлы
  • git commit -a(все файлы) -m(message-описание коммита)
  • git push - пушить на гит
  • git pull - взять с гита
